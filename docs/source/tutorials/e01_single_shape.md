# E01 — Single Shape

## Goal

Spawn one entity with a primitive shape component.

## What you’ll build

A minimal scene that renders one rectangle (or circle) centered on screen.

## Concepts taught

- entity creation
- position / size
- color
- draw call basics

## Acceptance criteria

- one entity appears in a stable position
- can change shape type via a constant or simple config

## Run

```bash
mini-arcade run --example 001_min_scene
```
