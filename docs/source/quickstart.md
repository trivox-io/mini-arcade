# Quickstart

## Users

```bash
pip install mini-arcade
mini-arcade --help
```

## Contributors (dev)

### 1) Install editable packages

```bash
# Windows (PowerShell)
.\scripts\dev_install.ps1
```

### 2) Run an example

```bash
mini-arcade run --example 001_min_scene
```

### 3) Run a game

```bash
mini-arcade run --game deja-bounce
```
