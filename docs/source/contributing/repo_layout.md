# Repo layout

```text
mini-arcade/
├─ packages/
│  ├─ mini-arcade/
│  ├─ mini-arcade-core/
│  ├─ mini-arcade-pygame-backend/
│  └─ mini-arcade-native-backend/
├─ games/
├─ examples/
└─ docs/
```

## Conventions

- `games/` are reference projects (they prove engine features).
- `examples/` are progressive tutorials (they become blog posts).
- `docs/` is the source of truth.
